单行文本转列
================

[原地址](https://askubuntu.com/questions/748524/how-to-convert-from-row-to-column)

### 问题

我有一个包含数字的文本文件，如下：

```bash
106849_01373    106849_01967    106850_00082    23025.7_01059
```

想转换成以下的形式：

```bash
106849_01373
106849_01967
106850_00082
23025.7_01059
```

我不知道要用什么命令，有人可以帮到我吗？

### 回答1

使用 `tr`，

```bash
tr -s '[:blank:]' '\n' <file.txt
```

示例

```bash
% cat file.txt                    
106849_01373    106849_01967    106850_00082    23025.7_01059   
106849_01373    106849_01967    106850_00082    23025.7_01059   
106849_01373    106849_01967    106850_00082    23025.7_01059   
106849_01373    106849_01967    106850_00082    23025.7_01059   

% tr -s '[:blank:]' '\n' <file.txt
106849_01373
106849_01967
106850_00082
23025.7_01059
106849_01373
106849_01967
106850_00082
23025.7_01059
106849_01373
106849_01967
106850_00082
23025.7_01059
106849_01373
106849_01967
106850_00082
23025.7_01059
```

### 回答2

使用 `xargs`：

```bash
xargs -n 1 < file.txt
```

示例：

```bash
$ cat file.txt                    
106849_01373    106849_01967    106850_00082    23025.7_01059 

$ xargs -n1 < file.txt
106849_01373
106849_01967
106850_00082
23025.7_01059
```

### 总结

* tr 使用了替换
* xargs -n 用于指定单行个数
